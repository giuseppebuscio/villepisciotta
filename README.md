# Ville Pisciotta

Sito web per Ville Pisciotta - Ville e appartamenti di lusso in Sicilia.

## Descrizione

Sito web moderno e responsive per presentare Villa MiraScopello e Villa Gelvi, due strutture di lusso situate in Sicilia.

## Tecnologie Utilizzate

- React 18
- React Router DOM
- Tailwind CSS
- Vite
- Lucide React (icone)

## Installazione

```bash
npm install
```

## Avvio in Sviluppo

```bash
npm run dev
```

## Build per Produzione

```bash
npm run build
```

## Preview Build

```bash
npm run preview
```

## Struttura del Progetto

- `/src/pages` - Pagine del sito (Home, Villa Mirascopello, Villa Gelvi, Contatti, ecc.)
- `/src/components` - Componenti riutilizzabili (Navbar, Footer, Hero, ecc.)
- `/public` - Risorse statiche (immagini, logo, manifest)
